# GradeUpperSecondary

GradeUpperSecondary represents a grade awarded to a student in upper secondary school (Swedish: gymnasieskola).
Grades are scoped to the organization of the API key used. The CourseCode field uses the national course code as defined by the Swedish school authorities (e.g. "MATMAT01a").
